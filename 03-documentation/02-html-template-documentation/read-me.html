3CSS Structure

This template includes 7 css files. Let's take a look closely at our stylesheet files:

  • style.css - common css styles for this template.
  • bootstrap.css - standard bootstrap css styles.
  • font-awesome.css - base styles for the font-awesome icons font.
  • layerslider.css- common css styles for LayerSlider
  • style-magnific-popup.css - contain css styles for the lightbox gallery

5Additional Info

Social Icons - please visit Font-Awesome for the complete list of social icons.

Fonts Used - I have used 1 type of Google Font in this template: Lato. If you would like to use different Google Fonts, you have to copy the font link in Google Fonts page and paste it in your index.html file.

Then you will need to add the new font name to your style.css file.
